# Grimm Tales

## Description

This is a text based rpg that is ran in the python terminal.

## Getting Started

### Dependencies

python 3.9

### Executing program

* To run the program download python 3.9 or better 
* then copy the code in code.py 
* finally paste the code into a new python file and press run
